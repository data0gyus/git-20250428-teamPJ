박건호 페이지

진행한 업무 목록

  • 레포지토리 만들기

    새로운 프로젝트를 시작하기 위해 GitHub에서 레포지토리를 생성하고 초기화 작업을 완료했습니다.

  • 허스키, 프리티어, ESLint 세팅

    코드 품질을 유지하기 위해 허스키(husky)로 Git hook을 설정하고, 프리티어(prettier)와 ESLint로 자동 포맷팅과 코드 품질 검사 기능을 추가했습니다.

  • 커밋 컨벤션 강제하는 커밋린트 세팅

    커밋 메시지 규칙을 강제하기 위해 CommitLint를 설정하여 일정한 규칙에 맞게 커밋 메시지를 작성하도록 유도했습니다.

  • 이슈 컨벤션 연결

    GitHub의 이슈 템플릿을 설정하여, 각 이슈를 생성할 때 규칙에 맞는 템플릿을 사용할 수 있도록 했습니다. 이를 통해 효율적인 이슈 관리가 가능해졌습니다.
